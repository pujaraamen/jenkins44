CI/CD Pipeline with Git & Jenkins

Automated Deployment to Jenkins Agents

Deployment Flow

Git Repository

Developer pushes code changes to Git.

➡

Jenkins

Webhook triggers Jenkins pipeline automatically.

➡

Build & Test

Jenkins builds the application and runs tests.

➡

Jenkins Agents

Deployment is executed on target agents/servers.

Pipeline Summary

  • Developer commits code to Git.
  • Git webhook triggers Jenkins job.
  • Jenkins checks out the latest code.
  • Build and validation steps run.
  • Application is deployed to Jenkins agents.
  • Nginx serves the static website.
